# 📘 Conhecendo a Linguagem JavaScript

Este repositório foi criado com o objetivo de praticar e demonstrar conceitos básicos da linguagem **JavaScript**, por meio de funções simples e didáticas.

É um projeto ideal para quem está iniciando na programação e quer fortalecer a lógica 🚀

---

## 🧠 Conteúdos abordados

Neste projeto, você encontrará exemplos práticos de:

- ✅ Estruturas condicionais (`if`, `else if`, `else`)
- ✅ Laços de repetição (`for`)
- ✅ Operadores:
  - Aritméticos (`%`)
  - Comparação (`===`)
  - Lógicos (`&&`)
- ✅ Funções em JavaScript
- ✅ Arrays (listas)
- ✅ Lógica matemática básica
- ✅ Uso de `console.log()`

---
